compyute.nn.trainer

Model trainer.

class compyute.nn.trainer.Trainer(model, optimizer, loss, metric=None, callbacks=None)[source]

Trainer utility used to train neural network models.

Parameters:

  • model (Module) – Model to be trained.

  • optimizer (_OptimizerLike) – Optimizer algorithm used to update model parameters based on gradients. See compyute.nn.optimizers for more details.

  • loss (_LossLike) – Loss function used to quantify the model performance. See Losses for more details.

  • metric (_MetricLike, optional) – Metric function used to evaluate the model. Defaults to None. See Metrics for more details.

  • callbacks (list[Callback], optional) – Callback functions that are executing each training epoch and step. Defaults to None. See compyute.nn.trainer.callbacks for more details.

evaluate_model(x, y, batch_size=32)[source]

Evaluates the model using a defined metric.

Parameters:

  • x (Tensor) – Input tensor.

  • y (Tensor) – Target tensor.

  • batch_size (int, optional) – Number of inputs processed in parallel. Defaults to 32.

Returns:

  • _ScalarLike – Loss value.

  • _ScalarLike, optional – Metric score.

train(x_train, y_train, epochs=100, val_data=None, batch_size=32)[source]

Trains the model using samples and targets.

Parameters:

  • X_train (Tensor) – Input tensor.

  • y_train (Tensor) – Target tensor.

  • epochs (int, optional) – Number of training iterations. Defaults to 100.

  • val_data (tuple[Tensor, Tensor], optional) – Data used for the validaton every epoch. Defaults to None.

  • batch_size (int, optional) – Number of inputs processed in parallel. Defaults to 32. If -1, all samples are used.
