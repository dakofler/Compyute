  • \(S_{out}\) … output sequence

  • in_channels (int) – Number of input channels.

  • out_channels (int) – Number of output channels (filters).

  • padding (_PaddingLike, optional) – Padding applied before convolution. Defaults to valid.

  • dilation (int, optional) – Dilation used for each axis of the filter. Defaults to 1.

  • bias (bool, optional) – Whether to use bias values. Defaults to True.

  • batchnorm_m (float, optional) – Momentum used in batch normalization. Defaults to 0.1.

  • dtype (DtypeLike, optional) – Datatype of weights and biases. Defaults to compyute.float32.

  • label (str, optional) – Module label. Defaults to None. If None, the class name is used.

class compyute.nn.Convolution1d(in_channels, out_channels, kernel_size, padding='valid', stride=1, dilation=1, bias=True, dtype=Dtype('float32'), label=None, training=False)[source]

Bases: Module

Applies a 1D convolution to the input for feature extraction.

\[y = b + \sum_{k=0}^{C_{in}-1} w_{k}*x_{k}\]

where \(*\) is the cross-correlation operator.

Shapes:

  • Input \((B, C_{in}, S_{in})\)

  • Output \((B, C_{out}, S_{out})\)

where

  • \(B\) … batch axis

  • \(C_{in}\) … input channels

  • \(S_{in}\) … input sequence

  • \(C_{out}\) … output channels

  • \(S_{out}\) … output sequence

Parameters:

  • in_channels (int) – Number of input channels.

  • out_channels (int) – Number of output channels (filters).

  • kernel_size (int) – Size of each kernel.

  • padding (_PaddingLike, optional) – Padding applied before convolution. Defaults to valid.

  • stride (int, optional) – Stride used for the convolution operation. Defaults to 1.

  • dilation (int, optional) – Dilation used for each axis of the filter. Defaults to 1.

  • bias (bool, optional) – Whether to use bias values. Defaults to True.

  • dtype (DtypeLike, optional) – Datatype of weights and biases. Defaults to compyute.float32.

  • label (str, optional) – Module label. Defaults to None. If None, the class name is used.

  • training (bool, optional) – Whether the module should be in training mode. Defaults to False.

Note

Weights are initialized from \(\mathcal{U}(-k, k)\), where \(k = \sqrt{\frac{1}{C_{in} * k * k}}\). Biases are initialized as zeros.

class compyute.nn.GRU(in_channels, h_channels, bias=True, activation='tanh', return_sequence=True, dtype=Dtype('float32'), label=None, training=False)[source]

Bases: Module

Gated Recurrent Unit module. For each element in the sequence the hidden state is computed as follows:

\[\begin{split}\begin{array}{ll} \\ r_t = \sigma(x_tW_{ir}^T + b_{ir} + h_{t-1}W_{hr}^T + b_{hr}) \\ z_t = \sigma(x_tW_{iz}^T + b_{iz} + h_{t-1}W_{hz}^T + b_{hz}) \\ n_t = act(x_tW_{in}^T + b_{in} + r_t \odot (h_{t-1}W_{hn}^T + b_{hn})) \\ h_t = (1 - z_t) \odot n_t + z_t \odot h_{t-1} \end{array}\end{split}\]

Note

This implementation follows the one by PyTorch which differs slightly from the original paper.

Shapes:

  • Input \((B, S, C_{in})\)

  • Output \((B, S, C_{h})\) if return_sequence=True else \((B, C_{h})\)

where

  • \(B\) … batch axis

  • \(S\) … sequence

  • \(C_{in}\) … input channels

  • \(C_{h}\) … hidden channels

Parameters:

  • in_channels (int) – Number of input channels.

  • h_channels (int) – Number of hidden channels.

  • bias (bool, optional) – Whether to use bias values. Defaults to True.

  • activation (Literal["relu", "tanh"], optional) – Activation function to use. Defaults to tanh.

  • return_sequence (bool, optional) – Whether to return the entire sequence or only the last hidden state.
