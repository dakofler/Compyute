Updates parameters following the Adam learning algorithm as described by Kingma et al., 2014.

Parameters:

  • parameters (Iterator[Parameter], optional) – Paramters to optimize. Defaults to None.

  • lr (float, optional) – Learning rate. Defaults to 1e-3.

  • beta1 (float, optional) – Exponential decay rate for the 1st momentum estimates. Defaults to 0.9.

  • beta2 (float, optional) – Exponential decay rate for the 2nd momentum estimates. Defaults to 0.999.

  • eps (float, optional) – Constant for numerical stability. Defaults to 1e-08.

  • weight_deyas (float, optional) – Weight decay factor. Defaults to 0.

  • grad_clip_value (float, optional) – Clips the gradient of all parameter to the set value. Defaults to None.
