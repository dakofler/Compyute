compyute.nn#

Function()

Neural network function base class.

FunctionCache()

Cache for intermediate data that needs to be cached for gradient computation.

no_caching()

Context manager to disable caching of values for gradient computation.

Losses#

Loss()

Loss base class.

BCELoss()

Computes the binary cross entropy loss from logits.

CrossEntropyLoss()

Computes the cross entropy loss from logits.

MSELoss()

Computes the mean squared error loss.

DiceLoss()

Computes the dice loss from logits as described by Milletari et al., 2016.

Metrics#

Metric()

Metric base class.

Accuracy()

Computes the accuracy score.

R2([eps])

Computes the coefficient of determination (R2 score).

Modules#

Module([label])

Neural network base module.

Identity([label])

Identity module that just forwards inputs and gradients.

ModuleList(modules)

List of modules.

Activations#

ReLU([label])

Applies the Rectified Linear Unit activation function to an input tensor.

LeakyReLU([alpha, label])

Leaky ReLu activation function.

GELU([label])

Gaussian Error Linear Unit activation function (using the \(tanh\) approximation) as described by Hendrycks et al., 2016.

FastGELU([label])

Sigmoid([label])

Sigmoid activation function.

SiLU([label])

Sigmoid Linear Unit activation function.

Softmax([label])

Softmax activation function.

Tanh([label])

Tanh activation function.

Blocks#

Conv1DBlock(in_channels, out_channels, ...)

Convolution block containing a 1D convolutional layer, followed by an optional batch normalization and an activation function.

Conv2DBlock(in_channels, out_channels, ...)

Convolution block containing a 2D convolutional layer, followed by an optional batch normalization and an activation function.

DenseBlock(in_channels, out_channels, activation)

Dense neural network block containing a linear transformation layer and an activation function.

Containers#

Sequential(*modules[, label])

Container that processes modules sequentially.

ParallelConcat(*modules[, concat_dim, label])

Container that processes modules in parallel and concatenates their outputs.

ParallelAdd(*modules[, label])

Container that processes modules in parallel and sums their outputs element-wise.

ResidualConnection(*modules[, ...])

Residual container implementing a residual connection around a block of modules.

Convolution#

Conv1D(in_channels, out_channels, kernel_size)

Applies a 1D convolution to the input for feature extraction.

Conv2D(in_channels, out_channels, kernel_size)

Applies a 2D convolution to the input for feature extraction.

ConvTranspose1D(in_channels, out_channels, ...)

Applies a 1D transposed convolution to the input for upsampling as descirbed by Long et al., 2015.

ConvTranspose2D(in_channels, out_channels, ...)

Embedding#

Embedding(n_embeddings, embedding_dim[, label])

Lookup embedding layer.

Linear#

Linear(in_channels, out_channels[, bias, label])

Applies a linear transformation to the input.

Normalizations#

BatchNorm1D(channels[, eps, m, label])

Implements Batch Normalization as described by Ioffe et al., 2015.

BatchNorm2D(channels[, eps, m, label])

LayerNorm(normalized_shape[, eps, label])

Implements Layer Normalization as described by Ba et al., 2016.

RMSNorm(normalized_shape[, eps, label])

Implements Root Mean Square Layer Normalization as described by Zhang et al., 2019.

Pooling#

MaxPooling2D([kernel_size, label])

Pooling layer used for downsampling where the maximum value within the pooling window is used.

AvgPooling2D([kernel_size, label])

Pooling layer used for downsampling where the average value within the pooling window is used.

Upsample2D([scaling, target_shape, label])

Layer used for upsamling by repeating values over the last two dimensions.

Recurrent#

GRU(in_channels, h_channels[, bias, ...])

Gated Recurrent Unit module as described by Cho et al., 2014.

LSTM(in_channels, h_channels[, bias, ...])

Long Short-Term Memory module as described by Hochreiter et al., 1997.

Recurrent(in_channels, h_channels[, bias, ...])

Elman Recurrent module.

Regularization#

Dropout([p, label])

Dropout layer used for regularization as described by Srivastava et al., 2019.

Shape#

Flatten([label])

Flattes tensors not including the batch dimension.

Reshape(shape[, label])

Reshapes tensors.

Slice(slice[, label])

Slices tensors.

Parameter and Buffer#

Parameter(data)

Trainable neural network parameter.

Buffer(data)

Neural network buffer variable.
