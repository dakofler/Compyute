compyute#

Tensors#

tensor(data[, device, dtype])

Creates a tensor object from arbitrary data.

Tensor(data)

Tensor object used for storing multidimensional data.

Tensor Operations#

Creating and Combining Operations#

Functions to create new tensors or combine existing tensors.

arange(stop[, start, step, device, dtype])

Returns a tensor of evenly spaced values using a step size within a given interval .:math:[start, stop).

empty(shape, *[, device, dtype])

Returns an tensor with uninitialized values.

empty_like(x)

Returns a tensor based on a given other tensor with uninitialized values.

full(shape, value, *[, device, dtype])

Returns a tensor of a given shape filled with a specified value.

full_like(x, value)

Returns a tensor of a given shape filled with a specified value.

identity(n, *[, device, dtype])

Returns a diagonal tensor of shape (n, n).

linspace(start, stop, n, *[, device, dtype])

Returns a tensor of num evenly spaced values within a given interval \([start, stop]\).

ones(shape, *[, device, dtype])

Returns a tensor of a given shape filled with ones.

ones_like(x)

Returns a tensor based on a given other tensor filled with ones.

zeros(shape, *[, device, dtype])

Returns a tensor of a given shape filled with zeros.

zeros_like(x)

Returns a tensor based on a given other tensor filled with zeros.

Unary Operations#

Functions that operate on one tensor.

abs(x)

Computes the element-wise absolute value of a tensor.

clip(x[, min_val, max_val])

Returns a tensor with values clipped to min_value and max_value.

cos(x)

Computes the element-wise cosine of a tensor.

cosh(x)

Computes the element-wise hyperbolic cosine of a tensor.

exp(x)

Computes the element-wise exponential of a tensor.

fft1d(x[, n, dim])

Computes the 1D Fast Fourier Transform over a given dimension.

fft2d(x[, n, dims])

Computes the 2D Fast Fourier Transform over given dimensions.

histogram(x[, bins, binrange, density, weights])

Computes the histogram of a tensor.

ifft1d(x[, n, dim])

Computes the inverse 1D Fast Fourier Transform over a given dimension.

ifft2d(x[, s, dims])

Computes the inverse 2D Fast Fourier Transform over given dims.

is_nan(x)

Returns True if the element in a tensor is not a number.

log(x)

Computes the element-wise natural log of a tensor.

log2(x)

Computes the element-wise log base 2 of a tensor.

log10(x)

Computes the element-wise log base 10 of a tensor.

real(x)

Returns the real part of a complex tensor.

round(x, decimals)

Rounds tensor elements.

sech(x)

Computes the element-wise secant of a tensor.

sin(x)

Computes the element-wise sine of a tensor.

sinh(x)

Computes the element-wise hyperbolic sine of a tensor.

sqrt(x)

Computes the element-wise square root of a tensor.

tan(x)

Computes the element-wise tangent of a tensor.

tanh(x)

Computes the element-wise hyperbolic tangent of a tensor.

Multinary Operations#

Functions that operate on two or more tensors.

allclose(x1, x2, *[, rtol, atol])

Returns True if all elements in the tensor are True.

convolve1d_fft(x1, x2)

Computes the convolution of two tensors using FFT over their last dimension.

convolve2d_fft(x1, x2)

Computes the convolution of two tensors using FFT over their last two dimensions.

dot(x1, x2)

Computes the dot product of two tensors.

einsum(subscripts, *tensors)

Computes the Einstein summation.

inner(*tensors)

Computes the inner product of two or more tensors.

outer(*tensors)

Computes the outer product of two or more tensors.

tensorprod(tensors)

Computes the element-wise product of any number of two or more tensors over their first dim.

tensorsum(tensors)

Computes the element-wise sum of any number of two or more tensors over their first dim.

Reducing Operations#

Functions that aggregate tensors.

all(x[, dim, keepdims])

Returns True if all elements in the tensor are True over given dimensions.

any(x[, dim, keepdims])

Returns True if at least one element in the tensor is True over given dimensions.

mean(x[, dim, keepdims])

Computes the mean of tensor elements over given dimensions.

norm(x[, dim, keepdims])

Computes the norm of tensor elements over given dimensions.

prod(x[, dim, keepdims])

Computes the product of tensor elements over given dimensions.

std(x[, dim, keepdims])

Computes the standard deviation of tensor elements over given dimensions.

tensorprod(tensors)

Computes the element-wise product of any number of two or more tensors over their first dim.

tensorsum(tensors)

Computes the element-wise sum of any number of two or more tensors over their first dim.

sum(x[, dim, keepdims])

Computes the sum of tensor elements over given dimensions.

var(x[, dim, ddof, keepdims])

Computes the variance of tensor elements over given dimensions.

Shape Operations#

Functions that change tensor shapes.

append(x, values[, dim])

Returns a copy of the tensor with appended values.

broadcast_to(x, shape)

Broadcast a tensor to a new shape.

concat(tensors[, dim])

Returns a new tensor by joining a sequence of tensors along a given dimension.

diagonal(x)

Expands a tensor by turning the last dim into a diagonal matrix.

flatten(x)

Returns a flattened 1D-tensor.

flip(x[, dim])

Returns a tensor with flipped elements along given dim.

insert_dim(x, dim)

Returns a view of the tensor with an added dimension.

movedim(x, from_dim, to_dim)

Move dimensions of a tensor to new positions.

pad(x, padding)

Returns a padded tensor using zero padding.

pad_to_shape(x, shape)

Returns a padded tensor using zero padding that matches a specified shape.

permute(x, dims)

Permutes a tensor's dimensions.

pooling1d(x, window_size[, stride])

Returns a windowed view of a tensor across the last dim.

pooling2d(x, window_size[, stride])

Returns a windowed view of a tensor across the last two dimensions.

repeat1d(x, n)

Repeat elements of a tensor along the last dimension.

repeat2d(x, n)

Repeat elements of a tensor along the last two dimensions.

reshape(x, shape)

Returns a new view of the tensor of a given shape.

split(x, splits[, dim])

Returns a list of new tensors by splitting the tensor.

squeeze(x)

Returns a tensor with removed dimensions where dim=1.

stack(tensors[, dim])

Returns a new tensor by stacking a sequence of tensors along a given dimension.

tile(x, n_repeats, dim)

Repeat elements of a tensor.

transpose(x, dim1, dim2)

Transposes a tensor by swapping two dimensions.

Selecting Operations#

Functions that return a subset of a tensor.

argmax(x[, dim, keepdims])

Returns the indices of maximum values along a given dim.

get_diagonal(x[, d])

Extract a diagonal or construct a diagonal tensor.

max(x[, dim, keepdims])

Computes the maximum of tensor elements over a given dim.

maximum(x1, x2)

Computes the element-wise maximum of two tensors or a tensor and a scalar.

mean(x[, dim, keepdims])

Computes the mean of tensor elements over given dimensions.

min(x[, dim, keepdims])

Computes the minimum of tensor elements over a given dim.

minimum(x1, x2)

Computes the element-wise minimum of two tensors or a tensor and a scalar.

get_diagonal(x[, d])

Extract a diagonal or construct a diagonal tensor.

tril(x[, diag_index])

Returns the lower triangle of a tensor below the diagonal of the last two dimensions.

triu(x[, diag_index])

Returns the upper triangle of a tensor above the diagonal of the last two dimensions.

unique(x)

Returns the unique ordered values of the tensor.

Data Types#

Available data types are:

bool_

Boolean.

int8

Signed 8 bit integer.

int16

Signed 16 bit integer.

int32

Signed 32 bit integer.

int64

Signed 64 bit integer.

float16

16 bit floating point.

float32

32 bit floating point.

float64

64 bit floating point.

complex64

Complex 64 bit floating point.

complex128

Complex 128 bit floating point.

use_dtype(dtype)

Context manager to set the default dtype for new tensors.

cuda

GPU device.

Utilities:

set_default_device(device)
