compyute.nn.functional#

Activations#

gelu(x)

Applies the Gaussian Error Linear Unit function to an input tensor.

fast_gelu(x)

Applies the Gaussian Error Linear Unit function to an input tensor.

leaky_relu(x[, alpha])

Applies the leaky ReLU function to an input tensor.

relu(x)

Applies the Rectified Linear Unit activation function to an input tensor.

sigmoid(x)

Applies the sigmoid function to an input tensor.

silu(x)

Applies the Sigmoid Linear Unit activation function to an input tensor.

softmax(x)

Applies the softmax function over the last axis of an input tensor.

tanh(x)

Applies the hyperbolic tangent activationfunction to an input tensor.

Convolutions#

convolve1d(x, f[, b, padding, stride, dilation])

Computes the convolution of two tensors over their last axis.

dilate1d(x, dilation)

Dilates a tensor in its last axis.

pad1d(x, padding)

Pads a tensor in its last axis.

convolve2d(x, f[, b, padding, stride, dilation])

Computes the convolution of two tensors over their last two axes.

dilate2d(x, dilation)

Dilates a tensor in its last two axes.

pad2d(x, padding)

Pads a tensor in its last two axes.

upsample2d(x, scaling, shape)

Upsamples a tensor by repeating it's elements over the last two axes.

maxpooling2d(x[, kernel_size])

Performs max pooling over the last two axes.

avgpooling2d(x[, kernel_size])

Performs average pooling over the last two axes.

Embeddings#

embedding(x, embedding_table)

Performs lookup embedding on a tensor of indices.

Linear#

linear(x, w[, b])

Applies a linear transformation to the input.

Losses#

mean_squared_error(y_pred, y_true)

Computes the mean squared error loss.

binary_cross_entropy(y_pred, y_true)

Computes the binary cross entropy loss.

cross_entropy(y_pred, y_true)

Computes the cross entropy loss.

Metrics#

accuracy_score(y_pred, y_true)

Computes the accuracy score given model predictions and target values.

r2_score(y_pred, y_true[, eps])

Computes the coefficient of determination (R2 score).

Normalization#

batchnorm1d(x, rmean, rvar, w, b[, m, eps, ...])

Performs 1D batch normalization on a tensor.

batchnorm2d(x, rmean, rvar, w, b[, m, eps, ...])

Performs 2D batch normalization on a tensor.

layernorm(x, w, b[, eps])

Performs layer normalization on a tensor.

rmsnorm(x, w[, eps])

Performs RMS normalization on a tensor.

Recurrent#

recurrent(x, w_i, b_i, w_h, b_h[, activation])

Applies the Elman recurrent function to a tensor.

lstm(x, w_ii, b_ii, w_if, b_if, w_ig, b_ig, ...)

Applies the LSTM recurrent function to a tensor.

gru(x, w_ir, b_ir, w_iz, b_iz, w_in, b_in, ...)

Applies the GRU recurrent function to a tensor.

Regularization#

dropout(x[, p, training])

Randomly sets tensor values to zero.
