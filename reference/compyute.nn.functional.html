compyute.nn.functional#

Activations#

gelu(x)

Applies the Gaussian Error Linear Unit activation function to the input.

fast_gelu(x)

Applies the Gaussian Error Linear Unit activation function to the input.

leaky_relu(x[, alpha])

Applies the leaky ReLU function to an input tensor.

relu(x)

Applies the Rectified Linear Unit activation function to the input.

sigmoid(x)

Applies the sigmoid function to an input tensor.

silu(x)

Applies the Sigmoid Linear Unit activation function to the input.

softmax(x[, dim])

Applies the softmax activation function to the last dimension of an input tensor.

tanh(x)

Applies the hyperbolic tangent activation function to the input.

Convolutions#

conv1d(x, f[, b, padding, stride, dilation])

Computes the convolution of two tensors over their last dimension.

dilate1d(x, dilation)

Dilates a tensor in its last dimension.

pad1d(x, padding)

Pads a tensor in its last dimension.

conv2d(x, f[, b, padding, stride, dilation])

Computes the convolution of two tensors over their last two dimensions.

dilate2d(x, dilation)

Dilates a tensor in its last two dimensions.

pad2d(x, padding)

Pads a tensor in its last two dimensions.

conv_transpose1d(x, f[, b, padding, stride, ...])

Computes the transposed convolution of two tensors over their last dimension.

conv_transpose2d(x, f[, b, padding, stride, ...])

Computes the transposed convolution of two tensors over their last two dimensions.

Embeddings#

embedding(x, emb_table)

Performs lookup embedding on a tensor of indices.

Linear#

linear(x, w[, b])

Applies a linear transformation to the input.

Losses#

mse_loss(logits, targets)

Computes the mean squared error loss.

bce_loss(logits, targets)

Computes the binary cross entropy loss from logits.

cross_entropy_loss(logits, targets[, eta])

Computes the cross entropy loss from logits.

dice_loss(logits, targets)

Computes the dice loss from logits.

Metrics#

accuracy_score(y_pred, y_true)

Computes the accuracy score given model predictions and target values.

r2_score(y_pred, y_true[, eps])

Computes the coefficient of determination (R2 score).

Normalization#

batchnorm1d(x, rmean, rvar, w, b[, m, eps, ...])

Performs 1D batch normalization on a tensor.

batchnorm2d(x, rmean, rvar, w, b[, m, eps, ...])

Performs 2D batch normalization on a tensor.

layernorm(x, w, b[, eps])

Performs layer normalization on a tensor.

rmsnorm(x, w[, eps])

Performs RMS normalization on a tensor.

Pooling#

maxpooling2d(x[, kernel_size])

Performs max pooling over the last two dimensions.

avgpooling2d(x[, kernel_size])

Performs average pooling over the last two dimensions.

upsample2d(x[, scaling, target_shape])

Upsamples a tensor by repeating values over the last two dimensions.

Recurrent#

recurrent(x, w_i, b_i, w_h, b_h[, activation])

Applies the Elman recurrent function to a tensor.

lstm(x, w_ii, b_ii, w_if, b_if, w_ig, b_ig, ...)

Applies the LSTM recurrent function to a tensor.

gru(x, w_ir, b_ir, w_iz, b_iz, w_in, b_in, ...)

Applies the GRU recurrent function to a tensor.

Regularization#

dropout(x[, p, training])

Randomly sets tensor values to zero.
