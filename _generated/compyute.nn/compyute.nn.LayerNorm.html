compyute.nn.LayerNorm#

class compyute.nn.LayerNorm(normalized_shape, eps=1e-05, label=None)[source]#

Implements Layer Normalization as described by Ba et al., 2016.

\[y = w \cdot \frac{x - E[x]}{\sqrt{Var[x] + \epsilon}} + b\]

where \(E[x]\) and \(Var[x]\) are computed over feature dimensions specified by normalized_shape.

Shapes:

  • Input \((B, ...)\)

  • Output \((B, ...)\)

where

  • \(B\) … batch dimension

Parameters:

  • normalized_shape (_ShapeLike) – Shape of the normalized tensor.

  • eps (float, optional) – Constant for numerical stability. Defaults to 1e-5.

  • label (str, optional) – Module label. Defaults to None. If None, the class name is used.

Note

Weights are initialized as ones, biases as zeros.
