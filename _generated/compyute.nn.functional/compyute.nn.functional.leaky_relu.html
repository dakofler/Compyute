compyute.nn.functional.leaky_relu(x, alpha=0.01)[source]#

Applies the leaky ReLU function to an input tensor.

Parameters:

  • x (Tensor) – Input tensor.

  • alpha (float, optional) – Slope of the negative output. Defaults to 0.01.

Returns:

Output tensor.
