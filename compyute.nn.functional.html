compyute.nn.functional.temperature_softmax(x, temperature=1)[source]

Applies the softmax function with temperature over the last axis of an input tensor.

\[softmax(x) = \frac{\exp(\frac{x}{T})}{\sum_{i=1}^N \exp(\frac{x_i}{T})}\]
Parameters:

  • x (Tensor) – Input tensor.

  • temperature (float, optional) – Temperature scaling to be applied in the calculation. Defaults to 1.

Returns:

Output tensor.

Return type:

Tensor

Raises:

ValueError – If temperature is 0.
